Program Studi 01 · FST UPITRA

S1 Sistem Informasi

Jenjang: Sarjana (S-1)
Masa Studi: 4 Tahun (8 Semester)
Kampus: Surakarta
Aktif
Deskripsi Program

Program Studi S1 Sistem Informasi membekali mahasiswa kemampuan menganalisis, merancang, mengimplementasikan, dan mengelola sistem informasi berbasis komputer untuk mendukung proses bisnis dan pengambilan keputusan organisasi.

Kurikulum dirancang dengan konsep link & match bersama Triputra Group dan mitra industri, memastikan relevansi langsung dengan kebutuhan dunia kerja.

Program Studi 02 · FST UPITRA

S1 Informatika

Jenjang: Sarjana (S-1)
Masa Studi: 4 Tahun (8 Semester)
Kampus: Surakarta
Aktif
Deskripsi Program

Program S1 Informatika berfokus pada fondasi teoritis ilmu komputer yang kuat: algoritma, struktur data, kecerdasan buatan (AI), machine learning, grafika komputer, dan pengembangan game/multimedia.

Mahasiswa diarahkan untuk menjadi inovator berbasis riset yang mampu mengembangkan solusi teknologi orisinal dengan pendekatan ilmiah.

Program Studi 03 · FST UPITRA

S1 Software Engineering

Jenjang: Sarjana (S-1)
Masa Studi: 4 Tahun (8 Semester)
Kampus: Surakarta
Aktif
Deskripsi Program

Program S1 Rekayasa Perangkat Lunak (Software Engineering) berfokus pada perancangan, pengembangan, pengujian, dan pemeliharaan perangkat lunak berkualitas tinggi menggunakan metodologi modern.

Mahasiswa dilatih dengan Agile, Scrum, DevOps, dan praktik industri internasional — siap langsung berkontribusi di tim pengembangan profesional.

Kompetensi Utama
Full-Stack Web Development Mobile App Development Software Testing & QA Agile / Scrum DevOps & CI/CD Microservices Architecture API Design (REST / GraphQL) Cloud Deployment Version Control (Git) Software Architecture Clean Code Principles UI/UX Development
